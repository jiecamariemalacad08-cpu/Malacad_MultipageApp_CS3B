 🌐 Streamlit Portfolio App

📌 Project Overview

This is a multipage portfolio web application built using the *Streamlit Framework in Python*.  
It showcases personal information, skills, projects, contact details, chatbot interaction, and a simple game section with a clean and modern UI design.

The system is designed to be interactive, user-friendly, responsive, and visually appealing while demonstrating creativity and technical understanding.

---

 ✨ Features

 🏠 Home Page
- Personal introduction
- Dark mode toggle
- Profile section with image
- Welcome interaction with input and button

 👩‍💻 About Me
- Personal background
- Education details
- Goals and ambitions
- Inspirational quote section

⚡ Skills Page
- Skill progress bars
- Interactive multiselect favorite skills
- Modern card design

📂 Projects Page
- Project showcase
- Search functionality
- Favorite project selection

📞 Contact Page
- Contact form
- Social media links
- GitHub, Facebook, and Instagram links

🤖 Portfolio Chatbot
- Interactive chatbot
- Answers questions about name, skills, projects, and goals

🎮 Simple Game Page
- Fun mini-game related to portfolio content
- Adds uniqueness and creativity to the application

---

🛠 Technologies Used

- Python
- Streamlit
- HTML
- CSS
- GitHub
- Streamlit Community Cloud
